This package does time series modeling and analysis, which provides observation data for fault slip modeling.
It has three parts
1) Raw time seires downloading
2) Pre-seismic time series modeling
3) Post-seismic time series modeling


Time convention:
There are three types of time: mjd, str, and dyr.
For internal computation, mjd is used. str is used only for inhancing readability. dyr should be avoid.
